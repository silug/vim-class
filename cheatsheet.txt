:help ins-special-keys
----------------------

<Esc>, <C-[>    | Go back to Normal mode
<C-C>           | Go back to Normal mode (without triggering autocommands)
<C-O>           | Switch to normal mode for one command
<C-\><C-O>      | Switch to normal mode for one command without moving cursor
<C-@>           | Insert previously inserted text and leave Insert mode
<C-A>           | Insert previously inserted text
<C-R>           | Insert the contents of a register
<C-R><C-R>      | Insert the contents of a register (literally)
<C-R><C-O>      | Insert the contents of a register (no autoindent)
<C-R><C-P>      | Insert the contents of a register literally and fix indent
<C-T>           | Indent the current line
<C-D>           | Unindent the current line one shiftwidth
0<C-D>          | Unindent the current line
<C-V>, <C-Q>    | Insert a special character
<C-K>           | Insert digraph
<C-]>           | Trigger abbreviation
<C-E>           | Insert the character below the cursor
<C-Y>           | Insert the character above the cursor
<Insert>        | Toggle replace mode
<BS>, <C-H>     | Delete the character before the cursor
<Del>           | Delete the character under the cursor
<C-W>           | Delete the word before the cursor
<C-U>           | Delete all text on the current line before the cursor
<C-X>           | Enter completion mode
     <C-L>      | Complete with whole lines
     <C-N>      | Complete with words in the current file (searching forward)
     <C-P>      | Complete with words in the current file (searching backwards)
     <C-K>      | Complete with dictionary words
     <C-T>      | Complete with thesaurus words
     <C-I>      | Complete with words in the current and included files
     <C-]>      | Complete with tags
     <C-F>      | Complete with file names
     <C-D>      | Complete with definitions or macros
     <C-V>      | Complete with Vim command line
     <C-U>      | Complete with user defined completion
     <C-O>      | Complete with omni completion
     s          | Complete with spelling suggestions
     <C-E>      | Stop completion with original text
     <C-Y>      | Stop completion with current selection

:help registers
---------------

"a to "z        | Named registers (replace)
"A to "Z        | Named registers (append)
""              | Unnamed register (contains last delete or yank)
"0              | Most recent yank
"1 to "9        | Recent deletes or changes
"-              | Last delete less than one line
".              | Last inserted text
"/              | Last search pattern
":              | Last command line
"%              | Current file name
"#              | Alternate file name
"+              | Clipboard contents
"*              | X11 selection contents
"~              | Dropped text
"_              | Black hole register
"=              | Expression register
